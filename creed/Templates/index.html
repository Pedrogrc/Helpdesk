Sobre o Sistema


O sistema de helpdesk é utilizado para cadastrar os chamados de suporte ao cliente.
Para realizar a operação, favor clicar no menu de cadastro de chamados.
Para verificar os chamados já registrados no sistema, clique no menu de lista.
O sistema foi desenvolvido para ser usado diretamente por um cliente do SENAI.
